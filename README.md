# 🎨 Imane HADI - Portfolio Artistique

Ce projet est un site vitrine multi-pages développé dans le cadre d’un TP avancé en développement web.

## 🌟 Objectif
Créer un site professionnel pour présenter les œuvres d’une artiste peintre, vendre des tableaux et permettre aux clients de passer des commandes personnalisées.

## 🧩 Fonctionnalités
- 🏠 Page d’accueil immersive avec vidéo
- 👩‍🎨 Présentation de l’artiste
- 🖼️ Galerie d’œuvres avec images responsives
- 💰 Vente de tableaux avec prix en dirham (DH)
- 📦 Commande personnalisée
- 📊 Tableau de tarifs
- 📞 Formulaire de contact complet
- 📍 Intégration Google Maps

## 🛠️ Technologies utilisées
- HTML5
- CSS3 (responsive design)
- JavaScript
